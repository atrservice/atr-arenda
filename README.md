# АТР-СПЕЦАРЕНДА — Сайт аренды спецтехники

## 📋 Требования
- Node.js 18+ (проверка: `node -v`)
- npm или yarn
- Git

## 🛠️ Установка и запуск

### Шаг 1: Клонирование / создание проекта
```bash
# Создайте папку и перейдите в неё
mkdir atr-specarenda && cd atr-specarenda

# Инициализируйте Next.js проект (если начинаете с нуля)
npx create-next-app@latest . --typescript --tailwind=false --eslint --app --src-dir --import-alias "@/*"

# Или скопируйте файлы из этого репозитория

# Установка зависимостей:
npm install
# или
yarn install
